Notes from DEV team :
______________________
-To run the game find the executionable jar file named "sokoban".
-When youre done with a level that you couldn't succeed, after pressing "Back" button press "Start" button.
-The games you created are automatically saved to the levels file. Since the max. level capacity is 15 you 
can delete unwanted levels by deleting the txt file with the corresponding name. Ex: if you want to delete
a level called deleteMe you should look for the file deleteMe.txt;
-We highly recommend not deleting the default level which are 1,2 and 3.
-Another thing : For the best experience play this game in a Windows computer. Some color losses may happen.
